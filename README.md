## Run Locally

Clone the project

```bash
  git clone git@github.com:BaBa0525/issue-manager.git
```

Go to the project directory

```bash
  cd issue-manager
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm dev
```
